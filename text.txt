Step-by-Step Guide
Step 1: Set Up Social Login APIs (Google and Facebook)
To enable Google and Facebook logins, you need to create developer accounts and obtain OAuth credentials. These are free but require setup.

1.1: Google OAuth
Go to Google Cloud Console.
Create a new project (e.g., VibeZone).
Navigate to APIs & Services > Credentials.
Click Create Credentials > OAuth 2.0 Client IDs.
Select Web application and configure:
Authorized JavaScript origins: http://localhost:3000
Authorized redirect URIs: http://localhost:3000/api/auth/google/callback
Save the Client ID and Client Secret.
1.2: Facebook OAuth
Go to Facebook Developers.
Create a new app (e.g., VibeZone).
Navigate to Settings > Basic.
Add Facebook Login product from the dashboard.
Configure Settings > Facebook Login:
Valid OAuth Redirect URIs: http://localhost:3000/api/auth/facebook/callback
Save the App ID and App Secret.
